# Architecture Overview

## Components

### 1. Identity Layer
Engineers verify credentials off-chain.
Mapped to pseudonymous Algorand wallets.

### 2. Reputation System
Non-transferable tokens track reviewer accuracy and consistency.

### 3. Blueprint Submission
Builders upload blueprint hash (IPFS) and lock reward funds.

### 4. Review System
Engineers stake and submit:
- detected flaws
- severity scores

### 5. Consensus Engine
Weighted voting based on reputation determines valid flaws.

### 6. Reward Distribution
Atomic transfers distribute rewards and penalties automatically.
